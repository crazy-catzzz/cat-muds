# cat-muds

This is a Minecraft Fabric mod that adds mud and mud items to Minecraft!

## How to install

- Make sure you're using MC 1.16.5
- Install Fabric and Fabric API
- Download a release
- Put the `.jar` file in the `mods` folder in your minecraft install
